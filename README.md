# EstesioTech 📱🩺

**ESTESIÓMETRO DIGITAL**

O **EstesioTech** é um aplicativo Android nativo desenvolvido para auxiliar no diagnóstico e monitoramento de pacientes com Hanseníase.

O projeto integra software e hardware para substituir os estesiômetros manuais (filamentos de Semmes-Weinstein) por uma solução digital precisa. O app se conecta a um dispositivo baseado em ESP32 via **Bluetooth Low Energy (BLE)** para receber dados de sensibilidade em tempo real.

## 🚀 Funcionalidades

* **Conectividade BLE:** Escaneamento, conexão e leitura de dados (Notify) de dispositivos ESP32.
* **Chat de Depuração:** Interface em tempo real para visualização de dados brutos do sensor.
* **Gestão de Usuários:** Sistema completo de Login e Cadastro para profissionais de saúde.
* **Interface Moderna:** UI construída com Material Design e navegação intuitiva.

## 🛠️ Tecnologias

* **Linguagem:** Kotlin
* **Arquitetura:** Android Nativo (View System/XML)
* **Comunicação:** Bluetooth Low Energy (BLE/GATT)
* **UI:** Material Design Components, RecyclerView
* **Build:** Gradle (Kotlin DSL)

## 🎯 Objetivo

Digitalizar o teste de avaliação sensitiva, permitindo o armazenamento de histórico clínico e, futuramente, o uso de IA para prognósticos de comprometimento neural.
